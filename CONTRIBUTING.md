# Guía de Contribución

¡Gracias por tu interés en contribuir a PyTorch MX! Esta es una comunidad dedicada a impulsar el uso de PyTorch en México y Latinoamérica.

## Cómo Contribuir

### Reportar Issues

- Usa el sistema de issues de GitHub para reportar bugs o sugerir mejoras
- Describe claramente el problema o sugerencia
- Incluye pasos para reproducir bugs cuando sea aplicable
- Agrega capturas de pantalla si es relevante

### Contribuir con Código

1. **Fork el repositorio**
   - Haz un fork del proyecto a tu cuenta de GitHub
   - Clona tu fork localmente

2. **Crea una rama para tu contribución**
   ```bash
   git checkout -b feature/nombre-de-tu-feature
   ```

3. **Realiza tus cambios**
   - Escribe código limpio y bien documentado
   - Sigue las convenciones de estilo del proyecto
   - Agrega tests cuando sea apropiado

4. **Commit tus cambios**
   ```bash
   git commit -m "Descripción clara de los cambios"
   ```

5. **Push a tu fork**
   ```bash
   git push origin feature/nombre-de-tu-feature
   ```

6. **Crea un Pull Request**
   - Describe claramente qué cambios realizaste y por qué
   - Referencia cualquier issue relacionado

### Contribuir con Tutoriales

- Los tutoriales se organizan en tres niveles: `basic`, `intermediate`, y `advanced`
- Asegúrate de que el tutorial esté en español
- Incluye ejemplos prácticos y ejecutables
- Documenta todas las dependencias necesarias

### Contribuir con Labs

- Los labs son ejercicios prácticos para la comunidad
- Deben ser autocontenidos y bien documentados
- Incluye soluciones o guías cuando sea apropiado

## Código de Conducta

Por favor lee y sigue nuestro [Código de Conducta](CODE_OF_CONDUCT.md).

## Estilo de Código

- Sigue las convenciones de PEP 8 para código Python
- Usa nombres descriptivos para variables y funciones
- Comenta código complejo
- Mantén las líneas de código en menos de 100 caracteres cuando sea posible

## Proceso de Revisión

- Todos los pull requests serán revisados por los mantenedores
- Se pueden solicitar cambios antes de la aceptación
- Una vez aprobado, tu contribución será fusionada al proyecto

## Preguntas

Si tienes preguntas, no dudes en:
- Abrir un issue
- Contactar a los mantenedores
- Unirte a las discusiones de la comunidad

¡Gracias por contribuir a PyTorch MX!
