# Labs - Ejercicios Prácticos

Esta carpeta contiene ejercicios prácticos y proyectos para aplicar lo aprendido.

## Propósito

Los labs son ejercicios hands-on diseñados para:
- Practicar conceptos de PyTorch
- Resolver problemas reales
- Experimentar con diferentes técnicas
- Prepararse para proyectos del mundo real

## Estructura de Labs

Cada lab debe incluir:
- **Descripción del problema**: Qué se debe resolver
- **Objetivos de aprendizaje**: Qué se aprenderá
- **Dataset**: Datos a utilizar (o cómo obtenerlos)
- **Plantilla de código**: Estructura inicial
- **Solución de referencia**: (Opcional) Implementación de ejemplo

## Para Contribuidores

Al crear un lab:
- Hazlo autocontenido y ejecutable
- Incluye instrucciones claras en español
- Proporciona métricas de éxito claras
- Considera diferentes niveles de dificultad
